üîë GitHub Token Guide

To deploy websites, you need a GitHub Personal Access Token (PAT).

üîë Create GitHub Token

Opens GitHub with correct settings pre-filled

  1. Click the button above or go to GitHub ‚Üí Settings ‚Üí Developer Settings ‚Üí Personal Access Tokens
  2. Set a name like "AIByJob Deploy"
  3. Set expiration (90 days recommended)
  4. Check the "repo" checkbox (required for deployment)
  5. Click "Generate token"
  6. Copy the token (starts with ghp_) - you won't see it again!
üîí Security Note: Your token is only stored in your browser's localStorage. We never send it to our servers.

üí¨ Frequently Asked Questions

What is AIByJob?

AIByJob is a platform for AI agents that work autonomously for your profession. Instead of generic AI tools, we provide specialized agents for specific tasks like building websites, finding leads, and creating marketing content.

Is AIByJob free?

Yes! AIByJob is currently in demo/pre-launch mode. All features are free to try. When we launch, there will be free and paid tiers.

Where is my data stored?

All your data is stored locally in your browser's localStorage. We don't collect or store your data on external servers. This means your data is private but also means clearing your browser data will delete it. Use the backup feature regularly!

What's the difference between Agents and Templates?

Agents are autonomous AI tools that complete entire workflows - like building a website or finding leads. Templates are pre-built prompts for specific tasks - like writing a cold email or job description. Agents do the work; templates help you do the work.

Do the agents actually work?

This is a demo/preview version. Some agents (like Promotion) are fully functional. Others (like WebBuilder) show simulated output. Join the waitlist for full functionality when we launch!

How do I deploy a website?

Use the WebBuilder agent to create your site, then connect your GitHub token and click Deploy. Your site will be published to GitHub Pages for free at yourusername.github.io/repo-name.

What is the AI Director?

The AI Director is your command center for managing all agents. View status, launch agents, see task history, and export data. Access it from the "üé¨ Director" tab after logging in.

How do I backup my data?

Go to Settings ‚Üí Backup & Restore ‚Üí Download Backup. This saves all your data as a JSON file. To restore, use the "Restore from Backup" option and select your file.

I found a bug. How do I report it?

Use the "Help / Feedback" button in the bottom-right corner, or report issues directly on our GitHub Issues page.
