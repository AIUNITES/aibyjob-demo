Overview

The Lead Qualification Agent automates the tedious process of evaluating inbound leads, allowing your sales team to focus on high-value conversations. Using AI-powered analysis, it examines multiple data points to determine lead quality and intent.

Unlike simple lead scoring rules, this agent understands context. It can identify a startup CEO requesting a demo as high-priority even if the company is small, while flagging a large enterprise contact as lower priority if they're just doing research.

How It Works

1. Lead Capture

The agent monitors your configured lead sources: web forms, email inbox, chatbot conversations, and CRM entries. When a new lead arrives, it triggers the qualification workflow.

2. Data Enrichment

Using the lead's email domain, the agent fetches company information from public sources and your connected data providers. It builds a comprehensive profile including company size, industry, funding history, and tech stack.

3. Scoring

The lead is scored across multiple dimensions:

  • Fit Score - How well the company matches your ICP
  • Intent Score - Signals of buying intent from their message and behavior
  • Engagement Score - Prior interactions with your content and brand
  • Timing Score - Budget cycle, contract renewal, and urgency signals

4. Routing & Action

Based on the composite score, the agent takes action:

  • Hot Leads (80+) - Immediate Slack alert + calendar booking link
  • Warm Leads (50-79) - Added to SDR queue with context summary
  • Cold Leads (<50) - Enrolled in nurture sequence

Configuration

The agent is configured using a JSON file that defines scoring rules, routing logic, and integrations. Here's an example configuration:

{
  "agent": "lead-qualification",
  "version": "2.1.0",
  
  "sources": {
    "webForms": ["contact", "demo-request", "pricing"],
    "emailInbox": "leads@yourcompany.com",
    "crm": "hubspot"
  },
  
  "scoring": {
    "weights": {
      "companyFit": 0.35,
      "intentSignals": 0.30,
      "engagement": 0.20,
      "timing": 0.15
    },
    
    "companyFit": {
      "employeeCount": {
        "1-50": 10,
        "51-200": 25,
        "201-1000": 35,
        "1001+": 30
      },
      "industries": {
        "technology": 30,
        "finance": 25,
        "healthcare": 20,
        "other": 10
      }
    },
    
    "intentSignals": {
      "keywords": {
        "pricing": 20,
        "demo": 25,
        "compare": 15,
        "budget": 30,
        "timeline": 25
      }
    }
  },
  
  "routing": {
    "hot": {
      "threshold": 80,
      "action": "slack-alert",
      "channel": "#sales-alerts",
      "assignTo": "round-robin"
    },
    "warm": {
      "threshold": 50,
      "action": "add-to-queue",
      "queue": "sdr-outbound"
    },
    "cold": {
      "threshold": 0,
      "action": "nurture-sequence",
      "sequence": "awareness-drip"
    }
  },
  
  "integrations": {
    "crm": {
      "provider": "hubspot",
      "syncFields": ["score", "enrichment", "routing"]
    },
    "slack": {
      "workspace": "your-company",
      "alertChannel": "#sales-alerts"
    },
    "enrichment": {
      "provider": "clearbit",
      "fields": ["company", "person"]
    }
  }
}

Best Practices

  • Start with defaults - Use the pre-configured scoring model first, then tune based on your data.
  • Review weekly - Check the analytics dashboard to see which leads converted and adjust weights.
  • Train your team - Ensure SDRs understand the score meaning and don't override routing.
  • Connect all sources - The more data the agent has, the better it scores.
  • Set up alerts - Configure Slack/email alerts for hot leads so response time stays under 5 minutes.
